README

这是一个简单的人人对战井字棋游戏

游戏方法如下：
    棋盘分为9个区域：
             |     |     
          1  |  2  |  3  
        -----------------
             |     |     
          4  |  5  |  6  
        -----------------
             |     |     
          7  |  8  |  9  

    游戏时在这九个区域内画上 X 或者 O, 当 X 或者 O , 3个符号连成一直线时, 那一方就算胜利

    用户只要在窗口的最下方的输入框中输入区域的编号，然后按submit按钮即可

    默认 X 先手
    
生成的可执行文件为 .app 文件, 在MACOS环境下可直接使用
.app文件在 dist文件夹下
